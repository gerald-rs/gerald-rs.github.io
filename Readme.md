# Portfolio — Gérald Iarovana Ratsiraka

Ce projet est un site portfolio personnel développé en HTML, CSS et JavaScript pur, dans un objectif de sobriété, de lisibilité et de maîtrise complète du code.

Il permet de présenter mon parcours, mes compétences, mes projets et mes moyens de contact à travers une interface élégante, cohérente et professionnelle.

---

## Objectifs du projet

- Créer un portfolio professionnel clair et structuré
- Utiliser uniquement des technologies web fondamentales (HTML / CSS / JS)
- Garder un code compréhensible et maintenable, même pour un débutant
- Mettre en avant les projets sans redirection vers d’autres pages
- Proposer une expérience utilisateur fluide sur desktop et mobile
- Respecter les bonnes pratiques SEO de base (structure, ancres, lisibilité)

---

## Technologies utilisées

- HTML : structure sémantique du site
- CSS : design, responsive, animations légères
- JavaScript : interactions dynamiques

Aucun framework n'est utilisé volontairement afin de conserver un contrôle total sur le rendu et le fonctionnement du site.

---

## Fonctionnalités principales

### Navigation
- Menu fixe en haut de page
- Navigation interne par ancres
- Menu mobile interactif

### Téléchargement du CV
- Bouton “Télécharger CV”
- Téléchargement réel d’un fichier PDF
- Fonctionne sur desktop et mobile

### Section Projets
- Projets affichés sous forme de cartes
- Le bouton "Détails" :
  - n'ouvre aucune nouvelle page
  - agrandit la carte sélectionnée
  - affiche un contenu détaillé (texte et images)
- Une seule carte projet peut être ouverte à la fois

### Responsive design
- Adaptation automatique à tous les formats d’écran
- Mise en page optimisée pour mobile, tablette et ordinateur

### Expérience utilisateur
- Design sobre et lisible
- Animations discrètes
- Interface intuitive


## Lancer le projet

Ouvrir le fichier index.html dans un navigateur web ou utiliser un serveur local pour un rendu optimal.
